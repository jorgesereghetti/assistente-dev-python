# Estudo de Caso 1 - DSA AI Coder - Criando Seu Assistente de Programação Python, em Python

Este projeto foi estendido para incluir um frontend web customizado, moderno e minimalista (tema escuro) que roda com FastAPI e permite alternar entre múltiplos provedores de LLM (Groq, OpenAI e Gemini) inserindo suas respectivas chaves de API diretamente na interface.

O projeto agora possui duas opções de interface:
1. **Interface Web Premium (Nova):** Interface minimalista em HTML/CSS/JS com tema escuro e suporte a múltiplos provedores.
2. **Interface Streamlit (Original):** Versão original do curso DSA.

---

## Como Rodar a Nova Interface Web Premium (Recomendado)

### 1. Ativar o Ambiente Virtual
Ative o ambiente virtual conda criado para o projeto:
```bash
conda activate dsaec1
```

### 2. Instalar as Dependências do Servidor Web
Instale o FastAPI e o Uvicorn no seu ambiente:
```bash
pip install fastapi uvicorn httpx
```

### 3. Iniciar o Servidor
Execute o servidor FastAPI:
```bash
uvicorn app:app --reload
```

### 4. Acessar a Interface
Abra o navegador no endereço:
👉 **http://localhost:8000**

Na barra lateral da interface web, você poderá:
- Selecionar seu provedor (Groq, OpenAI ou Gemini).
- Escolher o modelo desejado.
- Inserir e salvar sua chave de API com segurança (salva localmente no navegador).
- Ajustar a temperatura da geração.

---

## Como Rodar a Interface Streamlit Original

### 1. Ativar o Ambiente Virtual
```bash
conda activate dsaec1
```

### 2. Instalar as Dependências Originais
```bash
pip install -r requirements.txt
```

### 3. Rodar o App
```bash
streamlit run dsa_assistente.py
```
